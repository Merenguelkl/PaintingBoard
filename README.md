## A simple Painting Board implemented in PyQt5

### Install packages

`pip install pyqt5`

`pip install PyQt5-tools`

### Get start

`python begin.py`

---

### Function

#### Basic 

Open/Save file

Pencil

Barrel

#### Advanced 

Deblur

Emboss

Reverse
